## xenodmrc
a minimal OpenBSD xenodm config 

### Setup 
```
# mv /etc/X11/xenodm /etc/X11/xenodm.bak
# mkdir -p /etc/X11/xenodm
# install -m 444 Xresources /etc/X11/xenodm/Xresources
# install -m 755 Xsetup_0 /etc/X11/xenodm/Xsetup_0
```
